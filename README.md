# Go-Inspired Compiler (Java)

Prototype of a compiler built in Java based on the syntax and structure of the Go language.

Technologies:

- Java

Features:

- Lexical analyzer
- Partial syntax analyzer (incomplete)
- Partial semantic analyzer (incomplete)

Getting Started;

- Open the .java files in any IDE and run the tokenizer for initial tests.

Notes:

Project was started for academic purposes and remains unfinished.
